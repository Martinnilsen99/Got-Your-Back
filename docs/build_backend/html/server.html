server.get_classification()¶

Get the latest classification for today. If the timestamp in the url matches the timestamp of the latest classification in the file, something has gone wrong while classifying.

Returns

Dictionary with latest classification if file found and not empty. {Error: “FileEmpty” | “FileNotFound”} if not. http.HTTPStatus: Response status code 200 if file found and not empty, else 507.

Return type

dict
